# Mr. Bubbles Car Wash

## Overview
A car wash booking website for Mr. Bubbles, featuring pricing packages, rewards program, location info, and booking functionality.

## Tech Stack
- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS + shadcn/ui components
- **Animations**: Framer Motion
- **Routing**: React Router DOM v6

## Project Structure
```
src/
  components/     # Reusable UI components
    ui/           # shadcn/ui base components
  hooks/          # Custom React hooks
  lib/            # Utility functions
  pages/          # Page components
  assets/         # Static assets (images, etc.)
```

## Key Components
- **HeroSection**: Landing hero with logo and CTAs
- **PricingSection**: Car wash packages and pricing (JHB market-based)
- **RewardsSection**: Loyalty program info
- **LocationSection**: Business location details
- **Footer**: Site footer

## Development
- Run `npm run dev` to start the development server on port 5000
- Run `npm run build` for production build

## Pricing Packages (Updated Feb 2026)
Based on Johannesburg car wash market research:
- Quick Bubble: R80 - Basic exterior wash
- Full Service: R180 - Interior + exterior
- Premium Detail: R280 - Deep clean with wax
- Royal Treatment: R450 - Full detailing with engine clean
